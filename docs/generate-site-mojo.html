wildscribe:generate-site

Full name:

org.jboss.wildscribe:wildscribe-maven-plugin:3.1.0.Final:generate-site

Description:

A simple plugin used to generate a Wildscribe site.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.

Required Parameters

Name Type Since Description
<jbossHome> String - The application server's home directory. This is a required parameter and must point to a valid server directory. The server must include the module JAR's in order to get the log messages for the generated site.
User Property: wildscribe.jboss.home
Alias: jboss-home
