Generate Site Examples

Generates Wildscribe site for a WildFly server.

Generate Site Example

The example below shows how to generate a Wildscribe site:

<project>
    ...
    <build>
        ...
        <plugins>
            ...
            <plugin>
                <groupId>org.jboss.wildscribe</groupId>
                <artifactId>wildscribe-maven-plugin</artifactId>
                <version>2.0.0.Beta3-SNAPSHOT</version>
                <configuration>
                    <jboss-home>target/WildFly</jboss-home>
                    <site-dir>target/generated-docs/wildscribe</site-dir>
                </configuration>
            </plugin>
            ...
        </plugins>
        ...
    </build>
...
</project>
